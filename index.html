Aberystwyth Car Enthusiasts

iOS and Android Application
What does it do?

The A.C.E app allows members to quickly access the society's calendar so that nobody is in any doubt as to when and qwhere evemnts are taking place. It allows committee members to publish events and make changes to them retrospectively. Members can recieve notifcations when events are published or cancelled, along with logging in to see other members of the society and being able to show what they drive and see what others drive as well.

Why was it built?

Over the time I have been a member of A.C.E, event details have always been published on social media or in group chats. Both of these methods mean scrolling back to find details of events which can be infuriating somethimes. This app gives members a dedicated place where they can view events, along with all of the important details about them.

How does it work?

The main functionality of this app is done using Google Firebase. The app connects to a Firebase Firestore database to collect all the event data, along with cloud storage for the posters and profile pictures. The app uses listeners to display this data along with realtime updates if the data is changed. Firebase Authentication is used to handle the user base, and Cloud Functions are used to supply notifications to those devices which are opted in for these.

Cook Times Calculator

iOS Application
What does it do?

This app provides a quick way to convert the instructions on food packaging to air fryer instructions along with calculating the times it would take to roast a meat joint.

Why was it built?

Since air fryers have taken off, it's difficult to find somebody without one, and yet most food packaging still didn't contain instructions for cooking using an air fryer, conversion charts exist on the interent but this app can be accessed more quickly than those websites and does all the calculations for you.

How does it work?

Common practise for converting to air fryer conversions is to take 20 degrees off the temperature and reducing the time by 20%, once the user has input the original instructions and cooking method, the app then calculates the new instructions using these formulas and displays the results to the user.

The meat roasting section of the app works with the user enetring the weight of their joint and the type of meat it is, the app then calculates the time it would take this to roast using the time tasken per kilo * the number of kilos, depening on the meat selected, the user may have the choice of how well they would like the meat cooked.

Anagram Solver

iOS and Android Application
What does it do?

The Anagram Solver app is a simple application, built natively for both Android and iOS, designed to assist with solving anagrams and crossword puzzles. The app can full anagrams where all letters are known but the order is not, it can find potential solutions to anagrams where some of the letters are known but the order is not and can also help solve crossword puzzles where some of the letters are known and the position of each letter within the word is also know, beyond that, this app can also be used to find sub-anagrams, words of lesser length that can be made with some of the original letters.

Why was it built?

The intention behind this application was to built a simple app, where the focus was mainly set upon the publishing and review process of each of the major app store, the complexity of the app was less important here as this was an excersise in learning to publish an application and what was involved with that.

How does it work?

This app takes the anagram the user has entered and begins to loop through all words in the dictionary, frstly eliminating all words whose length does not match that of the anagram.

Once a word of the right length has been found, the algorithm loops through that word checking if the count of letter within the anagram is equal to the count of each letter within the current dictionary word. As soon as a letter count is found to be different between the anagram and dictionary word, the loop breaks as the dictionary word cannot possibly be a potential solution.

If the letter counts of each letter in the word match those in the anagram, then the word must be a potential solution and is appended to the array of results and presented to the user once the dictionary has been looped all the way through.
